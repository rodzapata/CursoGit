Configuracion inicial de Git

git config --global user.name "Rodney Zapata"

git config --global user.email "rodneyzapatapalacio@gmail.com"

git config --global core.editor "code --wait"

git config --global -e

Configuracion del Enter en Windows

git config --global core.autocrlf true"

Configuracion del Enter en Linuex y Mac

git config --global core.autocrlf input"

Comandos de transacciones

Agregamos un archivo al area de Stage

git add index.html

Verificamos el estado del repositorio Status

git status

Confirmamos los cambios realizado con Commit

git commit -m "Plantilla inicial"
